## Overview

This project demonstrates an end-to-end clinical trial analysis workflow using a synthetic Phase III clinical trial dataset.

The objective was to compare treatment outcomes between two investigational antihypertensive drugs and a placebo while demonstrating reproducible healthcare data analysis using R.

The workflow includes data preparation, exploratory data analysis (EDA), statistical hypothesis testing, logistic regression modelling, and publication-quality visualisations.

## Key Features

* Reproducible statistical workflow using Quarto
* Data cleaning and validation
* Exploratory data analysis (EDA)
* Logistic regression modelling
* Non-parametric statistical testing
* Publication-quality visualisations using ggplot2
